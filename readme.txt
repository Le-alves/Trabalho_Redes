Informações importantes sobre este trabalho: 

Resumo das Tarefas:
Acesso à IA generativa: Garantir o acesso a uma IA generativa via API.

Desenvolvimento da Aplicação:

Servidor:
Receber conexões dos clientes.
Funcionar em dois modos:
Automático: Respostas geradas pela IA.
Controlado: Um usuário escolhe entre a resposta da IA ou uma resposta escrita por ele.
Enganar o cliente para fazê-lo pensar que a resposta é de um humano.
Configurações iniciais para definir o tipo de execução e o tempo de espera antes de enviar a resposta da IA.

Cliente:
O usuário deve se registrar com seu nome.
Fazer perguntas ao servidor.
Decidir se a resposta recebida é de um humano ou da IA.
Informar o usuário se sua escolha estava correta.
Perguntar ao usuário se deseja continuar ou encerrar.
Mostrar estatísticas de respostas (IA vs. humanas) e o desempenho do usuário.
Controle de Respostas:

O servidor deve manter um histórico de perguntas e respostas, e o desempenho dos usuários.
Sugestão de usar arquivos para armazenar os dados.


Desafio Opcional: Implementar uma interface gráfica para facilitar o uso das aplicações, tanto no cliente quanto no servidor, para ganhar até 3 pontos extras.
